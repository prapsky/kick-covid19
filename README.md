# kick-covid19
Gives data related to Covid-19 filtered by Country and Date.

## Endpoint

The endpoint is:

    /api/v1/covid19/{cc}/{date}
    
For example we want to get data about Indonesia at March 24th, 2020:

    /api/v1/covid19/id/2020-03-24
    
We can get country code at [List of Internet top-level domains](https://en.wikipedia.org/wiki/List_of_Internet_top-level_domains).
